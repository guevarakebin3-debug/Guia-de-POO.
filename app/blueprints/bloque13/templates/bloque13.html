Bloque 13: Manejo de errores con try/except

¿Qué es el manejo de errores?

El manejo de errores permite que el programa no se detenga cuando ocurre una excepción. En Python usamos try para ejecutar código y except para capturar errores. 


# Ejemplo básico de manejo de errores
try:
    numero = int("abc")  # Esto genera un ValueError
    print(numero)
except ValueError:
    print("Error: no se pudo convertir a número")

# Salida: Error: no se pudo convertir a número

Ejercicios resueltos

Ejercicio 1: División entre cero


try:
    resultado = 10 / 0
    print(resultado)
except ZeroDivisionError:
    print("Error: división entre cero")

# Salida: Error: división entre cero

Ejercicio 2: Capturar múltiples excepciones


try:
    lista = [1, 2, 3]
    print(lista[5])  # IndexError
except ValueError:
    print("Error de valor")
except IndexError:
    print("Error: índice fuera de rango")

# Salida: Error: índice fuera de rango

Ejercicio 3: Uso de finally


try:
    archivo = open("datos.txt", "r")
    contenido = archivo.read()
    print(contenido)
except FileNotFoundError:
    print("Error: archivo no encontrado")
finally:
    print("Bloque finally ejecutado siempre")

# Salida:
# Error: archivo no encontrado
# Bloque finally ejecutado siempre
