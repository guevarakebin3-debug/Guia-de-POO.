Bloque 5: Condicionales if, elif, else

¿Qué son las estructuras condicionales?

Las estructuras condicionales permiten ejecutar diferentes bloques de código según se cumpla o no una condición. En Python usamos if, elif y else. 


# Ejemplo básico de condicional
edad = 18

if edad >= 18:
    print("Eres mayor de edad")   # Se ejecuta porque edad es 18
else:
    print("Eres menor de edad")

Ejercicios resueltos

Ejercicio 1: Clasifica un número como positivo, negativo o cero


numero = -3

if numero > 0:
    print("El número es positivo")
elif numero < 0:
    print("El número es negativo")
else:
    print("El número es cero")

# Salida: El número es negativo

Ejercicio 2: Verifica si un estudiante aprueba o reprueba


nota = 7

if nota >= 7:
    print("Aprobado")
else:
    print("Reprobado")

# Salida: Aprobado

Ejercicio 3: Determina el rango de edad


edad = 25

if edad < 12:
    print("Niño")
elif edad < 18:
    print("Adolescente")
elif edad < 65:
    print("Adulto")
else:
    print("Adulto mayor")

# Salida: Adulto
