Bloque 0: Introducción a la Programación Orientada a Objetos

¿Qué es la POO?

La Programación Orientada a Objetos (POO) organiza el software alrededor de objetos que representan entidades del mundo real.

Clase vs Objeto

Una Clase es el molde o plantilla que define atributos y métodos. Un Objeto es la instancia real creada a partir de esa clase. 


# Definimos una clase llamada Cliente
class Cliente:
    def __init__(self, nombre):
        # El constructor recibe un nombre y lo guarda en el atributo del objeto
        self.nombre = nombre

# Creamos un objeto (instancia) de la clase Cliente
cli = Cliente("Daniel")

# Mostramos el atributo nombre del objeto creado
print(cli.nombre)  # Salida: Daniel

Ejercicio 2: Crea la clase Persona con nombre y edad. Instancia 3 objetos diferentes.


# Definimos la clase Persona con dos atributos: nombre y edad
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

# Creamos tres objetos distintos de la clase Persona
p1 = Persona("Ana", 25)
p2 = Persona("Luis", 30)
p3 = Persona("María", 22)

# Mostramos los atributos de cada objeto
print(p1.nombre, p1.edad)  # Salida: Ana 25
print(p2.nombre, p2.edad)  # Salida: Luis 30
print(p3.nombre, p3.edad)  # Salida: María 22
