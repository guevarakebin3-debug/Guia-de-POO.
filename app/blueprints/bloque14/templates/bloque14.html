Bloque 14: Unpacking (Desempaquetado)

Concepto

El unpacking consiste en extraer valores de una estructura (lista, tupla, diccionario) y asignarlos a variables individuales.

Básico


a, b = [1, 2]
print(a, b)  # 1 2

a, b, c = (1, 2, 3)
print(a, b, c)  # 1 2 3

Con operador *


primero, *resto, ultimo = [1, 2, 3, 4, 5]
print(primero)  # 1
print(resto)    # [2, 3, 4]
print(ultimo)   # 5

En funciones


def suma(a, b, c):
    return a + b + c

numeros = [1, 2, 3]
print(suma(*numeros))  # 6

dict1 = {"a": 1}
dict2 = {"b": 2}
combinado = {**dict1, **dict2}
print(combinado)  # {'a': 1, 'b': 2}

En bucles


coordenadas = [(1, 2), (3, 4), (5, 6)]
for x, y in coordenadas:
    print(f"x={x}, y={y}")

# Salida:
# x=1, y=2
# x=3, y=4
# x=5, y=6

Ejercicios resueltos

Ejercicio 1: Desempaquetar tupla


primera, *mitad, ultima = (10, 20, 30, 40)
print(primera)  # 10
print(mitad)    # [20, 30]
print(ultima)   # 40

Ejercicio 2: Usar *lista en función


def multiplicar(a, b, c):
    return a * b * c

lista = [2, 3, 4]
print(multiplicar(*lista))  # 24

Ejercicio 3: Combinar diccionarios


dict1 = {"x": 1}
dict2 = {"y": 2}
dict3 = {**dict1, **dict2}
print(dict3)  # {'x': 1, 'y': 2}
