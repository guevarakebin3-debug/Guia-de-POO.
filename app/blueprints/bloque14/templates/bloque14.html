Bloque 14: Manejo de archivos en Python

¿Qué es el manejo de archivos?

Python permite trabajar con archivos de texto para guardar y leer información. Se usa la función open() junto con modos como "r" (leer), "w" (escribir), y "a" (agregar). 


# Abrimos un archivo en modo escritura
archivo = open("ejemplo.txt", "w")

# Escribimos contenido en el archivo
archivo.write("Hola Denis, este es un archivo de prueba.")

# Cerramos el archivo
archivo.close()

Ejercicios resueltos

Ejercicio 1: Leer el contenido de un archivo


# Abrimos el archivo en modo lectura
archivo = open("ejemplo.txt", "r")

# Leemos todo el contenido
contenido = archivo.read()
print(contenido)  # Salida: Hola Denis, este es un archivo de prueba.

archivo.close()

Ejercicio 2: Escribir varias líneas en un archivo


archivo = open("datos.txt", "w")

archivo.write("Primera línea\n")
archivo.write("Segunda línea\n")
archivo.write("Tercera línea\n")

archivo.close()

Ejercicio 3: Usar with para manejar archivos automáticamente


# El bloque with cierra el archivo automáticamente
with open("datos.txt", "r") as archivo:
    for linea in archivo:
        print(linea.strip())

# Salida:
# Primera línea
# Segunda línea
# Tercera línea
