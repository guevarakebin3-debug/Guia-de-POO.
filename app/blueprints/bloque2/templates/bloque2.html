Bloque 2: Variables y Tipos de Datos

¿Qué son las variables?

Una variable es un espacio en memoria que almacena un valor. En Python no necesitas declarar el tipo explícitamente, se asigna automáticamente según el valor.

Tipos de datos básicos en Python

  • int: números enteros (ej. 10, -5)
  • float: números decimales (ej. 3.14, -0.5)
  • str: cadenas de texto (ej. "Hola")
  • bool: valores lógicos (True o False)

# Asignamos diferentes tipos de valores a variables
numero_entero = 10       # int
numero_decimal = 3.14    # float
texto = "Hola Denis"     # str
es_valido = True         # bool

# Mostramos los valores y sus tipos
print(numero_entero, type(numero_entero))   # Salida: 10 
print(numero_decimal, type(numero_decimal)) # Salida: 3.14 
print(texto, type(texto))                   # Salida: Hola Denis 
print(es_valido, type(es_valido))           # Salida: True

Ejercicios resueltos

Ejercicio 1: Declara una variable para tu nombre y otra para tu edad


nombre = "Denis"
edad = 28

print("Mi nombre es", nombre)  # Salida: Mi nombre es Denis
print("Tengo", edad, "años")   # Salida: Tengo 28 años

Ejercicio 2: Convierte un número entero a decimal


numero = 5
numero_decimal = float(numero)

print(numero_decimal)  # Salida: 5.0

Ejercicio 3: Explica la diferencia entre int y float

int representa números enteros sin decimales, mientras que float representa números con decimales. Ejemplo: 10 es int, 10.0 es float.
