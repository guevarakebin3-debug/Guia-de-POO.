Bloque 9: Tuplas en Python

¿Qué son las tuplas?

Una tupla es una colección ordenada de elementos, similar a una lista, pero inmutable (no se puede modificar después de creada). Se definen con paréntesis (). 


# Definimos una tupla de números
numeros = (1, 2, 3, 4, 5)

# Mostramos la tupla completa
print(numeros)  # Salida: (1, 2, 3, 4, 5)

# Accedemos al primer elemento (índice 0)
print(numeros[0])  # Salida: 1

# Intentar modificar un elemento genera error
# numeros[0] = 10  # TypeError: 'tuple' object does not support item assignment

Ejercicios resueltos

Ejercicio 1: Crear una tupla de frutas y acceder a sus elementos


frutas = ("manzana", "banana", "naranja")

print(frutas[1])   # Salida: banana
print(frutas[-1])  # Salida: naranja

Ejercicio 2: Convertir una lista en tupla


lista = [10, 20, 30]
tupla = tuple(lista)

print(tupla)  # Salida: (10, 20, 30)

Ejercicio 3: Uso de tuplas en asignación múltiple


# Asignación múltiple usando tupla
coordenadas = (4, 7)
x, y = coordenadas

print("x:", x)  # Salida: x: 4
print("y:", y)  # Salida: y: 7
