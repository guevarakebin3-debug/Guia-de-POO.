Bloque 15: Funciones de orden superior

Concepto

Una función de orden superior es aquella que recibe otra función como parámetro o devuelve una función. En Python, ejemplos comunes son map(), filter() y reduce().

map()

Aplica una función a cada elemento de una colección.


numeros = [1, 2, 3, 4]
cuadrados = list(map(lambda x: x ** 2, numeros))
print(cuadrados)  # [1, 4, 9, 16]

filter()

Filtra elementos según una condición.


pares = list(filter(lambda x: x % 2 == 0, numeros))
print(pares)  # [2, 4]

reduce()

Reduce la colección a un único valor acumulado.


from functools import reduce

suma = reduce(lambda x, y: x + y, numeros)
print(suma)  # 10

Combinación


numeros = [1, 2, 3, 4, 5, 6]

resultado = list(
    map(lambda x: x ** 2,
        filter(lambda x: x % 2 == 0, numeros))
)
print(resultado)  # [4, 16, 36]

Ejercicios resueltos

Ejercicio 1: Incrementar con map()


valores = [2, 4, 6]
resultado = list(map(lambda x: x + 1, valores))
print(resultado)  # [3, 5, 7]

Ejercicio 2: Filtrar mayores a 3


valores = [1, 2, 3, 4, 5]
resultado = list(filter(lambda x: x > 3, valores))
print(resultado)  # [4, 5]

Ejercicio 3: Multiplicar con reduce()


from functools import reduce

valores = [1, 2, 3, 4]
resultado = reduce(lambda x, y: x * y, valores)
print(resultado)  # 24
