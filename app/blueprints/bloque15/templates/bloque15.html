Bloque 15: Creación de clases y objetos

¿Qué es una clase?

Una clase es un molde o plantilla que define las características (atributos) y comportamientos (métodos) de un objeto.

¿Qué es un objeto?

Un objeto es una instancia de una clase, es decir, un ejemplar concreto creado a partir de esa plantilla. 


# Definimos una clase Persona
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        print("Hola, mi nombre es", self.nombre)

# Creamos un objeto de la clase Persona
p1 = Persona("Ana", 25)

# Usamos sus atributos y métodos
print(p1.nombre)   # Salida: Ana
print(p1.edad)     # Salida: 25
p1.saludar()       # Salida: Hola, mi nombre es Ana

Ejercicios resueltos

Ejercicio 1: Crear una clase Coche


class Coche:
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo

    def mostrar_info(self):
        print("Marca:", self.marca, "| Modelo:", self.modelo)

# Creamos un objeto
c1 = Coche("Toyota", "Corolla")
c1.mostrar_info()  # Salida: Marca: Toyota | Modelo: Corolla

Ejercicio 2: Crear varios objetos de la misma clase


p1 = Persona("Luis", 30)
p2 = Persona("María", 22)

p1.saludar()  # Salida: Hola, mi nombre es Luis
p2.saludar()  # Salida: Hola, mi nombre es María

Ejercicio 3: Explica la diferencia entre clase y objeto

La clase es la definición o plantilla, mientras que el objeto es una instancia concreta creada a partir de esa clase.
