Bloque 11: Conjuntos en Python

Concepto

Los conjuntos (set) son colecciones sin duplicados y sin orden garantizado. Se usan para eliminar repetidos y realizar operaciones matemáticas como unión e intersección. 


numeros = {1, 2, 2, 3, 3}
print(numeros)  
# Salida: {1, 2, 3} → elimina duplicados

Operaciones matemáticas


A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)  # Unión → {1,2,3,4,5,6}
print(A & B)  # Intersección → {3,4}
print(A - B)  # Diferencia → {1,2}
print(A ^ B)  # Diferencia simétrica → {1,2,5,6}

Métodos principales


conjunto = {1, 2, 3}
conjunto.add(4)       # Agregar
conjunto.remove(2)    # Eliminar (error si no existe)
conjunto.discard(5)   # Eliminar sin error
print(conjunto)       # {1, 3, 4}

Ejercicios resueltos

Ejercicio 1: Operaciones básicas


A = {1, 2, 3}
B = {3, 4, 5}

print("Unión:", A | B)          # {1,2,3,4,5}
print("Intersección:", A & B)   # {3}
print("Diferencia:", A - B)     # {1,2}

Ejercicio 2: Eliminar duplicados de una lista


lista = [1, 2, 2, 3, 3, 3, 4]
sin_dup = list(set(lista))
print(sin_dup)  # [1,2,3,4]

Ejercicio 3: Diferencia simétrica


A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

resultado = (A | B) - (A & B)
print(resultado)  # {1,2,5,6}
