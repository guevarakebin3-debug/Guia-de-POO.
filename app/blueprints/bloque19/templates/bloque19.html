Bloque 19: Relaciones UML en código

1. Asociación

Concepto: Una clase usa a otra, pero no depende de ella para existir.


class Profesor:
    def __init__(self, nombre):
        self.nombre = nombre

class Curso:
    def __init__(self, titulo, profesor):
        self.titulo = titulo
        self.profesor = profesor  # asociación

p = Profesor("Ana")
c = Curso("Matemáticas", p)
print(c.profesor.nombre)  # Ana

2. Agregación

Concepto: Una clase contiene objetos de otra, pero los objetos pueden existir sin la clase contenedora.


class Estudiante:
    def __init__(self, nombre):
        self.nombre = nombre

class Escuela:
    def __init__(self, nombre):
        self.nombre = nombre
        self.estudiantes = []  # agregación

    def agregar_estudiante(self, estudiante):
        self.estudiantes.append(estudiante)

e1 = Estudiante("Luis")
escuela = Escuela("Colegio Central")
escuela.agregar_estudiante(e1)
print(escuela.estudiantes[0].nombre)  # Luis

3. Composición

Concepto: Una clase crea y controla la vida de los objetos que contiene. Si la clase se destruye, también sus objetos.


class Motor:
    def __init__(self, tipo):
        self.tipo = tipo

class Auto:
    def __init__(self, modelo):
        self.modelo = modelo
        self.motor = Motor("Gasolina")  # composición

a = Auto("Toyota")
print(a.motor.tipo)  # Gasolina

4. Herencia UML

Concepto: Una clase hija hereda atributos y métodos de la clase padre.


class Persona:
    def __init__(self, nombre):
        self.nombre = nombre

class Estudiante(Persona):
    def __init__(self, nombre, carrera):
        super().__init__(nombre)
        self.carrera = carrera

e = Estudiante("Carlos", "Ingeniería")
print(e.nombre, e.carrera)

Ejercicios resueltos

Ejercicio 1: Asociación

Crear clase Libro y clase Autor, donde un libro tiene un autor asociado.


class Autor:
    def __init__(self, nombre):
        self.nombre = nombre

class Libro:
    def __init__(self, titulo, autor):
        self.titulo = titulo
        self.autor = autor

a = Autor("Gabriel García Márquez")
l = Libro("Cien años de soledad", a)
print(l.autor.nombre)  # Gabriel García Márquez

Ejercicio 2: Agregación

Crear clase Departamento que contenga empleados.


class Empleado:
    def __init__(self, nombre):
        self.nombre = nombre

class Departamento:
    def __init__(self, nombre):
        self.nombre = nombre
        self.empleados = []

    def agregar_empleado(self, empleado):
        self.empleados.append(empleado)

d = Departamento("TI")
d.agregar_empleado(Empleado("Ana"))
print(d.empleados[0].nombre)  # Ana

Ejercicio 3: Composición

Crear clase Casa que contenga objetos Habitacion creados internamente.


class Habitacion:
    def __init__(self, nombre):
        self.nombre = nombre

class Casa:
    def __init__(self):
        self.habitaciones = [Habitacion("Sala"), Habitacion("Cocina")]

c = Casa()
for h in c.habitaciones:
    print(h.nombre)
# Salida:
# Sala
# Cocina
