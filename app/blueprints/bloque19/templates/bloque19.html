Bloque 19: Abstracción en POO

¿Qué es la abstracción?

La abstracción consiste en definir métodos que deben ser implementados por las clases hijas, sin especificar su funcionamiento en la clase base. En Python se logra usando abc (Abstract Base Classes). 


from abc import ABC, abstractmethod

# Clase abstracta
class Figura(ABC):
    @abstractmethod
    def area(self):
        pass

# Clase concreta que implementa el método
class Circulo(Figura):
    def __init__(self, radio):
        self.radio = radio

    def area(self):
        return 3.14 * self.radio ** 2

c1 = Circulo(5)
print("Área del círculo:", c1.area())  # Salida: 78.5

Ejercicios resueltos

Ejercicio 1: Clase abstracta Animal


from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def sonido(self):
        pass

class Perro(Animal):
    def sonido(self):
        return "Guau"

class Gato(Animal):
    def sonido(self):
        return "Miau"

animales = [Perro(), Gato()]
for a in animales:
    print(a.sonido())

# Salida:
# Guau
# Miau

Ejercicio 2: Clase abstracta Vehículo


class Vehiculo(ABC):
    @abstractmethod
    def mover(self):
        pass

class Bicicleta(Vehiculo):
    def mover(self):
        return "Avanza con pedales"

class Coche(Vehiculo):
    def mover(self):
        return "Avanza con motor"

v1 = Bicicleta()
v2 = Coche()
print(v1.mover())  # Avanza con pedales
print(v2.mover())  # Avanza con motor

Ejercicio 3: Importancia de la abstracción

La abstracción permite definir contratos claros, asegurando que todas las clases hijas implementen los métodos necesarios, lo que mejora la consistencia y organización del código.
