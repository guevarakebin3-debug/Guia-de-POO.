Bloque 6: Bucles while y for

¿Qué son los bucles?

Los bucles permiten repetir un bloque de código varias veces. En Python tenemos dos principales: while y for.

Bucle while

El bucle while repite mientras la condición sea verdadera. 


contador = 1

while contador <= 5:
    print("Iteración", contador)
    contador += 1

# Salida:
# Iteración 1
# Iteración 2
# Iteración 3
# Iteración 4
# Iteración 5

Bucle for

El bucle for recorre una secuencia (lista, cadena, rango, etc.). 


for numero in range(1, 6):
    print("Número:", numero)

# Salida:
# Número: 1
# Número: 2
# Número: 3
# Número: 4
# Número: 5

Ejercicios resueltos

Ejercicio 1: Suma de números del 1 al 10 usando while


suma = 0
i = 1

while i <= 10:
    suma += i
    i += 1

print("La suma es:", suma)  # Salida: La suma es: 55

Ejercicio 2: Recorre una lista de nombres usando for


nombres = ["Ana", "Luis", "María"]

for nombre in nombres:
    print("Hola", nombre)

# Salida:
# Hola Ana
# Hola Luis
# Hola María

Ejercicio 3: Imprime los números pares del 1 al 10


for numero in range(1, 11):
    if numero % 2 == 0:
        print(numero)

# Salida:
# 2
# 4
# 6
# 8
# 10
