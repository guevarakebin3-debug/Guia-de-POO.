Bloque 10: Diccionarios en Python

¿Qué son los diccionarios?

Un diccionario es una estructura de datos que almacena pares clave: valor. Se definen con llaves {} y permiten acceder a los valores usando sus claves. 


# Definimos un diccionario de estudiante
estudiante = {
    "nombre": "Ana",
    "edad": 22,
    "carrera": "Ingeniería"
}

# Accedemos a un valor usando la clave
print(estudiante["nombre"])   # Salida: Ana

# Agregamos una nueva clave-valor
estudiante["promedio"] = 9.1
print(estudiante)  # Salida: {'nombre': 'Ana', 'edad': 22, 'carrera': 'Ingeniería', 'promedio': 9.1}

Ejercicios resueltos

Ejercicio 1: Crear un diccionario de país


pais = {
    "nombre": "Ecuador",
    "capital": "Quito",
    "poblacion": 18_000_000
}

print(pais["capital"])  # Salida: Quito

Ejercicio 2: Recorrer un diccionario con for


for clave, valor in pais.items():
    print(clave, ":", valor)

# Salida:
# nombre : Ecuador
# capital : Quito
# poblacion : 18000000

Ejercicio 3: Verificar si una clave existe


if "capital" in pais:
    print("La clave 'capital' existe")  # Salida: La clave 'capital' existe
