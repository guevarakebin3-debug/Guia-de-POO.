Bloque 12: Excepciones en Python

¿Qué son las excepciones?

Una excepción ocurre cuando se produce un error en tiempo de ejecución. El manejo de excepciones permite que el programa no se detenga y pueda reaccionar al error.

Estructura completa


try:
    # Código que puede fallar
    numero = int("abc")
except ValueError as e:
    # Manejo del error
    print("Error de conversión:", e)
else:
    # Se ejecuta si NO hubo error
    print("Conversión exitosa")
finally:
    # Se ejecuta SIEMPRE
    print("Bloque finally ejecutado")

Tipos de errores comunes

  • ValueError → valor inválido (ej. int("abc"))
  • TypeError → tipo incorrecto (ej. str + int)
  • IndexError → índice fuera de rango
  • KeyError → clave inexistente en dict/set
  • ZeroDivisionError → división entre cero
  • FileNotFoundError → archivo no encontrado

Ejercicios resueltos

Ejercicio 1: Capturar ValueError


try:
    numero = int(input("Ingrese un número: "))
    print("Número ingresado:", numero)
except ValueError:
    print("Error: Debe ingresar un número válido")

Ejercicio 2: Capturar IndexError


lista = [1, 2, 3]
try:
    print(lista[5])
except IndexError:
    print("Error: índice fuera de rango")

Ejercicio 3: Manejar múltiples excepciones


try:
    num = int("abc")  # ValueError
    resultado = 10 / 0  # ZeroDivisionError
except ValueError:
    print("Error de conversión")
except ZeroDivisionError:
    print("Error: división entre cero")
