Bloque 3: Operadores en Python

Operadores aritméticos

Permiten realizar cálculos matemáticos básicos.


a = 20
b = 4

print(a + b)  # Suma → 24
print(a - b)  # Resta → 16
print(a * b)  # Multiplicación → 80
print(a / b)  # División decimal → 5.0
print(a // b) # División entera → 5
print(a % b)  # Módulo → 0
print(a ** b) # Potencia → 160000

Operadores de comparación

Comparan valores y devuelven True o False.


x = 10
y = 20

print(x == y)  # Igualdad → False
print(x != y)  # Diferente → True
print(x > y)   # Mayor → False
print(x < y)   # Menor → True
print(x >= 10) # Mayor o igual → True
print(y <= 20) # Menor o igual → True

Operadores lógicos

Permiten combinar condiciones.


cond1 = True
cond2 = False

print(cond1 and cond2) # AND → False
print(cond1 or cond2)  # OR → True
print(not cond1)       # NOT → False

== vs is

== compara contenido, is compara referencia en memoria.


a = [1, 2]
b = [1, 2]
c = a

print(a == b) # True → mismo contenido
print(a is b) # False → objetos distintos
print(a is c) # True → misma referencia

Precedencia de operadores

Orden en que Python evalúa las operaciones.


x = 2 + 1 * 2 % 2 + (2 ** 1) // 2
print(x)  # Resultado: 3
# Orden: ** → % → // → * → +
