Bloque 1: El constructor __init__

¿Qué es el constructor __init__?

El método __init__ es especial en Python y se ejecuta automáticamente cuando se crea un objeto. Sirve para inicializar los atributos de la clase. 


# Definimos una clase Persona
class Persona:
    def __init__(self, nombre, edad):
        # El constructor recibe parámetros y los asigna a atributos del objeto
        self.nombre = nombre
        self.edad = edad

# Creamos un objeto de la clase Persona
p1 = Persona("Ana", 25)

# Mostramos los atributos inicializados
print(p1.nombre)  # Salida: Ana
print(p1.edad)    # Salida: 25

Ejercicios resueltos

Ejercicio 1: Crea una clase Coche con atributos marca y modelo


class Coche:
    def __init__(self, marca, modelo):
        # Guardamos los valores recibidos en atributos
        self.marca = marca
        self.modelo = modelo

# Instanciamos un objeto de tipo Coche
c1 = Coche("Toyota", "Corolla")

# Mostramos los atributos inicializados
print(c1.marca)   # Salida: Toyota
print(c1.modelo)  # Salida: Corolla

Ejercicio 2: Explica qué sucede si no defines __init__

Si no defines __init__, Python crea un constructor por defecto que no inicializa atributos. Podrás crear objetos, pero no tendrán valores iniciales asignados automáticamente.
