Bloque 8: Listas en Python

¿Qué son las listas?

Una lista es una estructura de datos que permite almacenar múltiples valores en una sola variable. Se definen con corchetes [] y pueden contener distintos tipos de datos. 


# Definimos una lista de números
numeros = [1, 2, 3, 4, 5]

# Mostramos la lista completa
print(numeros)  # Salida: [1, 2, 3, 4, 5]

# Accedemos al primer elemento (índice 0)
print(numeros[0])  # Salida: 1

# Accedemos al último elemento
print(numeros[-1])  # Salida: 5

Ejercicios resueltos

Ejercicio 1: Agregar y eliminar elementos de una lista


frutas = ["manzana", "banana", "naranja"]

# Agregar un elemento al final
frutas.append("uva")
print(frutas)  # Salida: ['manzana', 'banana', 'naranja', 'uva']

# Eliminar un elemento específico
frutas.remove("banana")
print(frutas)  # Salida: ['manzana', 'naranja', 'uva']

Ejercicio 2: Recorrer una lista con for


colores = ["rojo", "verde", "azul"]

for color in colores:
    print("Color:", color)

# Salida:
# Color: rojo
# Color: verde
# Color: azul

Ejercicio 3: Uso de slicing (rebanado)


numeros = [10, 20, 30, 40, 50]

# Obtener los primeros 3 elementos
print(numeros[:3])  # Salida: [10, 20, 30]

# Obtener los últimos 2 elementos
print(numeros[-2:])  # Salida: [40, 50]
