Bloque 4: Entrada y salida en Python

Salida con print()

La función print() muestra información en pantalla. Admite f-strings, y parámetros como sep y end. 


nombre = "Ana"
edad = 25

print("Nombre:", nombre)  
# Salida: Nombre: Ana

print(f"Nombre: {nombre}, Edad: {edad}")  
# Salida: Nombre: Ana, Edad: 25

print("Python", "Java", sep=" - ", end=" !\n")  
# Salida: Python - Java !

Entrada con input()

La función input() lee datos del usuario. Siempre devuelve un str, por lo que es necesario convertirlo si se requiere otro tipo. 


nombre = input("Ingrese su nombre: ")
edad = int(input("Ingrese su edad: "))   # casting a int
altura = float(input("Ingrese su altura: ")) # casting a float

print(f"{nombre} tiene {edad} años y mide {altura} m")

Casting (conversión de tipos)

Se usa para transformar el texto leído en otro tipo de dato: int(), float(), str().

nombre = input("Nombre: ")

Ejercicio 2: Suma y promedio


num1 = int(input("Número 1: "))
num2 = int(input("Número 2: "))

suma = num1 + num2
promedio = suma / 2

print(f"Suma: {suma}, Promedio: {promedio}")

Ejercicio 3: Input sin conversión


numero = input("Ingrese un número: ")

# Se concatena como texto, no se suma.
