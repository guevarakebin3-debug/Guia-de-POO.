Bloque 7: Funciones en Python

¿Qué son las funciones?

Una función es un bloque de código reutilizable que realiza una tarea específica. Se define con la palabra clave def y puede recibir parámetros y devolver valores. 


# Definimos una función simple
def saludar():
    print("Hola, bienvenido a Python")

# Llamamos a la función
saludar()  # Salida: Hola, bienvenido a Python

Ejercicios resueltos

Ejercicio 1: Función con parámetros


def sumar(a, b):
    resultado = a + b
    return resultado

print(sumar(5, 3))  # Salida: 8

Ejercicio 2: Función que devuelve un valor


def cuadrado(numero):
    return numero ** 2

print(cuadrado(4))  # Salida: 16

Ejercicio 3: Función con valores por defecto


def presentar(nombre, edad=18):
    print("Nombre:", nombre)
    print("Edad:", edad)

presentar("Ana")        # Salida: Nombre: Ana / Edad: 18
presentar("Luis", 25)   # Salida: Nombre: Luis / Edad: 25
