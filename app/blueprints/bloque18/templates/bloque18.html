Bloque 18: Polimorfismo en POO

¿Qué es el polimorfismo?

El polimorfismo permite que diferentes clases implementen métodos con el mismo nombre, pero con comportamientos distintos. Esto facilita trabajar con objetos de distintas clases de manera uniforme. 


class Animal:
    def hablar(self):
        print("El animal hace un sonido")

class Perro(Animal):
    def hablar(self):
        print("El perro dice: ¡Guau!")

class Gato(Animal):
    def hablar(self):
        print("El gato dice: ¡Miau!")

# Uso polimórfico
animales = [Perro(), Gato(), Animal()]

for a in animales:
    a.hablar()

# Salida:
# El perro dice: ¡Guau!
# El gato dice: ¡Miau!
# El animal hace un sonido

Ejercicios resueltos

Ejercicio 1: Clase Figura con polimorfismo


class Figura:
    def area(self):
        pass

class Circulo(Figura):
    def __init__(self, radio):
        self.radio = radio

    def area(self):
        return 3.14 * self.radio ** 2

class Rectangulo(Figura):
    def __init__(self, ancho, alto):
        self.ancho = ancho
        self.alto = alto

    def area(self):
        return self.ancho * self.alto

figuras = [Circulo(5), Rectangulo(4, 6)]

for f in figuras:
    print("Área:", f.area())

# Salida:
# Área: 78.5
# Área: 24

Ejercicio 2: Polimorfismo con métodos comunes


class Vehiculo:
    def mover(self):
        print("El vehículo se mueve")

class Bicicleta(Vehiculo):
    def mover(self):
        print("La bicicleta avanza con pedales")

class Coche(Vehiculo):
    def mover(self):
        print("El coche avanza con motor")

vehiculos = [Bicicleta(), Coche()]

for v in vehiculos:
    v.mover()

# Salida:
# La bicicleta avanza con pedales
# El coche avanza con motor

Ejercicio 3: Importancia del polimorfismo

El polimorfismo permite diseñar sistemas flexibles y escalables, donde diferentes clases pueden responder a las mismas llamadas de métodos de manera específica.
