Bloque 18: Principios de POO y conceptos avanzados

1. Abstracción

Concepto: Mostrar lo esencial y ocultar lo innecesario. El usuario interactúa con métodos sin conocer la implementación interna.


class Calculadora:
    def sumar(self, a, b):
        return a + b

calc = Calculadora()
print(calc.sumar(5, 3))  # 8

2. Encapsulamiento

Concepto: Proteger datos controlando acceso a atributos mediante métodos.


class Cuenta:
    def __init__(self):
        self._saldo = 0  # atributo protegido

    def depositar(self, valor):
        self._saldo += valor

    def ver_saldo(self):
        return self._saldo

c = Cuenta()
c.depositar(100)
print(c.ver_saldo())  # 100

3. Herencia

Concepto: Reutilizar código en clases hijas que heredan atributos y métodos de la clase padre.


class Persona:
    def __init__(self, nombre):
        self.nombre = nombre

class Cliente(Persona):
    def __init__(self, nombre, correo):
        super().__init__(nombre)
        self.correo = correo

cli = Cliente("Daniel", "d@mail.com")
print(cli.nombre, cli.correo)

4. Polimorfismo

Concepto: Un mismo método puede tener distintos comportamientos según la clase que lo implemente.


class Animal:
    def sonido(self): pass

class Perro(Animal):
    def sonido(self): print("Ladra")

class Gato(Animal):
    def sonido(self): print("Maulla")

class Vaca(Animal):
    def sonido(self): print("Muge")

for animal in [Perro(), Gato(), Vaca()]:
    animal.sonido()
# Salida:
# Ladra
# Maulla
# Muge

5. Property

Concepto: Controlar acceso a atributos con validación usando @property y @setter.


class Producto:
    def __init__(self, nombre, precio):
        self._nombre = nombre
        self._precio = precio

    @property
    def precio(self):
        return self._precio

    @precio.setter
    def precio(self, valor):
        if valor < 0:
            raise ValueError("Precio inválido")
        self._precio = valor

p = Producto("Laptop", 900)
p.precio = 1200
print(p.precio)  # 1200

6. Clases abstractas

Concepto: Definen contratos de comportamiento que deben implementarse en clases hijas.


from abc import ABC, abstractmethod

class Figura(ABC):
    @abstractmethod
    def area(self): pass

class Triangulo(Figura):
    def __init__(self, base, altura):
        self.base = base
        self.altura = altura
    def area(self):
        return (self.base * self.altura) / 2

t = Triangulo(10, 5)
print(t.area())  # 25.0

Ejercicio 1: Abstracción

Crear clase Triangulo que implemente area() de una clase abstracta Figura.


from abc import ABC, abstractmethod

class Figura(ABC):
    @abstractmethod
    def area(self): pass

class Triangulo(Figura):
    def __init__(self, base, altura):
        self.base = base
        self.altura = altura
    def area(self):
        return (self.base * self.altura) / 2

t = Triangulo(10, 5)
print(t.area())  # 25.0
