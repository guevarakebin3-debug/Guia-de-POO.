Bloque 16: Archivos y JSON

Modos de apertura

  • "r" → lectura
  • "w" → escritura (sobrescribe)
  • "a" → append (agrega al final)
  • "x" → crear; error si existe

Lectura y escritura de texto


# Escribir en archivo
with open("archivo.txt", "w") as f:
    f.write("Hola mundo\n")
    f.write("Segunda línea\n")

# Leer todo el contenido
with open("archivo.txt", "r") as f:
    contenido = f.read()
    print(contenido)

# Leer línea por línea
with open("archivo.txt", "r") as f:
    for linea in f:
        print(linea.strip())

JSON (JavaScript Object Notation)

JSON es un formato de intercambio de datos basado en pares clave-valor. En Python se maneja con el módulo json. 


import json

# Guardar en JSON
datos = {"nombre": "Juan", "edad": 25, "activo": True}
with open("datos.json", "w") as f:
    json.dump(datos, f, indent=2)

# Cargar desde JSON
with open("datos.json", "r") as f:
    cargado = json.load(f)
    print(cargado["nombre"])  # Juan

# Lista de objetos
usuarios = [
    {"nombre": "Ana", "edad": 20},
    {"nombre": "Luis", "edad": 30}
]
with open("usuarios.json", "w") as f:
    json.dump(usuarios, f)

Ejercicio 2: Guardar y cargar JSON


import json

datos = {"x": 10, "y": 20}
with open("datos.json", "w") as f:
    json.dump(datos, f)

with open("datos.json", "r") as f:
    cargado = json.load(f)
    print(cargado)  # {'x': 10, 'y': 20}
