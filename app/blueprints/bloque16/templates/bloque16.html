Bloque 16: Encapsulamiento en POO

¿Qué es el encapsulamiento?

El encapsulamiento consiste en ocultar los detalles internos de una clase y controlar el acceso a sus atributos y métodos. En Python se logra usando convenciones como:

  • atributo → Público
  • _atributo → Protegido (convención)
  • __atributo → Privado (name mangling)

class CuentaBancaria:
    def __init__(self, titular, saldo):
        self.titular = titular      # Público
        self._saldo = saldo         # Protegido
        self.__pin = "1234"         # Privado

    def mostrar_saldo(self):
        print("Saldo:", self._saldo)

    def validar_pin(self, pin):
        if pin == self.__pin:
            print("Acceso permitido")
        else:
            print("Acceso denegado")

cuenta = CuentaBancaria("Denis", 1000)

# Acceso público
print(cuenta.titular)  # Salida: Denis

# Acceso protegido (convención: no debería usarse fuera de la clase)
print(cuenta._saldo)   # Salida: 1000

# Acceso privado (genera error)
# print(cuenta.__pin)  # AttributeError

Ejercicios resueltos

Ejercicio 1: Crear una clase con atributo privado


class Usuario:
    def __init__(self, nombre, password):
        self.nombre = nombre
        self.__password = password

    def validar(self, intento):
        return intento == self.__password

u1 = Usuario("Ana", "secreto")
print(u1.validar("secreto"))  # True
print(u1.validar("1234"))     # False

Ejercicio 2: Usar métodos para acceder a atributos privados


class Producto:
    def __init__(self, nombre, precio):
        self.nombre = nombre
        self.__precio = precio

    def get_precio(self):
        return self.__precio

    def set_precio(self, nuevo_precio):
        if nuevo_precio > 0:
            self.__precio = nuevo_precio

p1 = Producto("Laptop", 1200)
print(p1.get_precio())  # 1200
p1.set_precio(1500)
print(p1.get_precio())  # 1500

Ejercicio 3: Explica la importancia del encapsulamiento

El encapsulamiento protege los datos de accesos indebidos, mantiene la integridad de la información y permite controlar cómo se modifican los atributos de una clase.
